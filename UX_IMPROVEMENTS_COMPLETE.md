## 🎯 РЕАЛИЗОВАННЫЕ УЛУЧШЕНИЯ

### 1. ✅ Система "Одно активное информационное окно"

**Архитектура:**
```python
# Кэш: {(user_id, chat_id): (user_command_id, bot_response_id)}
self._last_info_messages: dict[tuple[int, int], tuple[int, int]] = {}
```

**Логика:**
- При новой информационной команде удаляется предыдущая пара (команда + ответ)
- Удаление только по замещению, НЕ по таймеру
- Информационное окно живёт до следующего запроса

**Участвуют:**
- `/help` (помощь, старт)
- `/профиль` (profile, проф)
- `/баланс` (balance, бал, деньги, бабки)
- `/ачивки` (achievements, ачивы, достижения)
- `/топ` (top, рейтинг, лидеры)
- `/топнеделя` (weekly)
- `/семьи` (families, браки)
- `/брак` (marriage)

**НЕ участвуют (игровые события):**
- Ферма, казино, дуэли, клады, свадьбы, пидор, пара
