## Лимитки

- Канон стоимости: магазинная цена 50★-подарка 700 × 1.5 = **1050** (sell 735).
  НЕ от формулы 50★×10.
- Каждая из 8 лимиток — отдельная награда (свой item_code, вес, вероятность,
  статистика, событие в ленте). ×2/×3 — та же лимитка кратностью (min_qty=qty).
- Источники (сумма всех 8 ID на 1M открытий): 🏺 Коллекционер ~264 000 (главный),
  💎 Джекпот ~116 000 (второй), 🎯🌾🐣 ~4 600–4 900 (редкая удача).

## Статистика (внедрено)

`GET /internal/cases/stats?case=<item_code>` (бот, по секрету) отдаёт на каждый
кейс: число открытий, потрачено ешек, сколько Premium / лимиток / джекпотов
выпало, и последние 10 крупных выпадений. Источник — леджер `case_openings`
(агрегаты в `app/repositories/cases.py::get_case_stats`).

## Социальные события (внедрено)

`app/features/cases/rare_drop.py` шлёт в общий чат по утверждённым шаблонам:
- лимитка → «🎁 РЕДКАЯ НАХОДКА … выбил <предмет> из кейса <кейс> · Стоимость: 1050 ешек»;
- Premium → «⭐ PREMIUM DROP … выбил Premium N месяцев из кейса <кейс>»;
- денежный джекпот → «💎 ДЖЕКПОТ … сорвал джекпот · 25000 ешек».
