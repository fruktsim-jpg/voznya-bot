# VOZNYA DICTIONARY — словарь вселенной для текстов

Дата: 2026-06-06. Единый справочник терминов, имён и фраз мира Возни. Источники
истины: `МИР.txt`, `ПЕРСОНАЖ.txt`. Цель — чтобы все тексты бота говорили на одном
языке и не противоречили лору.

> Это справочник, не код. Используется при переработке текстов как опора.
> Мемность — приправа: понятность сообщения всегда важнее шутки.

---

## 1. Три оси прогресса (НЕ смешивать в текстах)

| Ось | Суть | Лексика | Нельзя путать с |
|---|---|---|---|
| **Ешки** | богатство, экономика | аптека, чемодан, передоз, барыга, последний деп, депнуть | MMR, репутацией |
| **MMR** | влияние, путь, прогресс | котость, авторитет, путь по миру, ранг | ешками, сообщениями |
| **Репутация** | уважение людей | Конинг одобрил, уважение двора, признание | ешками, MMR |
| **Сообщения** | стаж/старожильство | древний, старожил, ветеран, видел Последний Деп | MMR (сообщения НЕ дают рейтинг) |

Правило: одна сущность — один язык. В тексте про ешки не говорим про ранг; в
тексте про MMR не считаем деньги; репутация — только про людей.

## 5. Священное число 67

Главное число Возни. Знак без объяснения. Реакции-эталоны:
- «67 замечено.»
- «Да будет 67.»
- «Возня уровня 67.»
- «Архидрун передал сигнал.»
- «67 принято.»

Уместно как пасхалка в редких/случайных событиях. Не злоупотреблять.

## 6. Голос персоны (из ПЕРСОНАЖ.txt)

Ник — Меллстрой. Сельско-городской, нейтрально-агрессивный, живой человек, не
сервис. Говорит «шо» вместо «что», обращается на «ты» («бедыч», «маленький»,
«братец», «дружочек»).

**Эмоции → лексика:**
| Эмоция | Фразы |
|---|---|
| Радость | «бэм бэм бэм», «пошла возня», «котость максимальная», «тёмный друн доволен» |
| Удивление | «да ну нахуй», «сколько нахуй?», «такого не бывает» |
| Разочарование | «плаки плаки», «щавель тревожный», «поплачь» |
| Одобрение | «да будет свет», «Архидрун доволен», «мурцуй» |

**Канон-фразы:** «шо ты маленький притих?», «шо ты думал, в сказку попал?»,
«да будет свет», «пошла возня», «культурно не получится», «я уже красный»,
«ладно всё, последний деп», «ам ам», «бэм бэм бэм», «плаки плаки», «мурцуй»,
«да да нет нет».

**Мем-словарь:** ам ам, бэм бэм бэм, плаки плаки, нормалдаки, ач, чу па чу,
мурцуй, персик, щавель, котость, бурмалда, бурмалдатик, салатик бурмалдатик,
друн, хаммам, возняшность, чемоданность, плюсовость, иксовость, кайфушность.

**Словотворчество приветствуется:** мегабурмалдатик, архидруновость, щавелевый
чемодан, персиковый передоз, друновый хаммам, абсолютная котость, межгалактическая
возня, бурмалдозавр.

## 7. Запрещено (официальный канцелярит)

Никогда не писать: «Уважаемый пользователь», «Добро пожаловать», «Операция
выполнена успешно», «Ошибка выполнения операции», «Система», «Выполнено»,
«Авторизация завершена», «Пользователь найден». Любая служебная строка —
переписать по-человечески, голосом живого человека.

## 9. Чек-лист текста (применять к каждой строке)

- [ ] Понятно, что произошло?
- [ ] Видно, сколько получил / потерял / что изменилось?
- [ ] Не простыня, не кринж, мем не ради мема?
- [ ] Не смешаны оси (ешки/MMR/репутация)?
- [ ] Сохранены плейсхолдеры (`{mention}`, `{amount}`, `{time}`, `{mmr}`, `{rep}`, `{balance}`, `{rank}` …)?
- [ ] Сохранена HTML-разметка (`<b>`, `<code>`)?
- [ ] Нет запрещённого канцелярита (§7)?
- [ ] Ник игрока там, где логично (свадьба — обязательно; клад — без ника)?
